Cartão Giratório

Explore a rotação interativa deste Web Component acessível e personalizável, projetado para oferecer uma experiência visual elegante e funcional.

Cartão Simples

Este cartão utiliza apenas a tag <cartao-giratorio>, sem imagens, exibindo cores padrão. Possui todas as funcionalidades de acessibilidade, incluindo navegação por teclado, suporte a leitores de tela e indicação de interatividade. Trata-se de um exemplo do que é exibido quando imagens não são utilizadas.

Cartão com Acessibilidade

Este cartão incorpora imagens personalizadas e preserva todas as funcionalidades de acessibilidade, incluindo navegação por teclado, atributos ARIA, anúncios para leitores de tela e uma indicação visível de interatividade, semelhante ao exemplo anterior.

Cartão Decorativo

Este cartão utiliza o atributo decorative para atuar como um elemento visual, sem semântica para leitores de tela. Preserva a indicação de interatividade e os controles por teclado para usuários que enxergam, além de oferecer interação por mouse/toque e rotação automática.
